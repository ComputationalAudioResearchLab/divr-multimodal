# Analysis

This folder stores test-set reports generated by `python -m src --test`.

The default report location is `analysis/qwen2audio_lora/<task_name>/` and contains:

- `predictions.csv`
- `summary.json`
- `confusion_matrix.csv`
- `confusion_matrix.png`
- `accuracy_by_label.csv`
- `accuracy_by_label.png`
